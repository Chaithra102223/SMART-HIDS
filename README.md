# SMART-HIDS
SMART HOST-BASED INTRUSION DETECTION AND FIREWALL MONITORING SYSTEM

## SMART-HIDS is a web-based application designed to monitor, analyze, and detect suspicious activities occuring within a host system in real time.

### Features
- User Authentication
- Real-Time Dashboard
- Intrusion Detection
- Firewall Monitoring
- Network Traffic Analysis
- Threat Alerts
- Activity Logs
- Data Visualization

### Technologies Used
- Python
- HTML
- CSS
- JavaScript
- SQLite
- Ubuntu Linux
